# Research: One Thing to Try This Week (Closing)

## Recommended Tool: Ollama with `ollama launch`
- Fastest getting-started path of all tools covered
- Free, open source, runs locally
- One command: `ollama launch claude` sets up everything
- Requires Ollama v0.15+ (latest is v0.15.3, released Feb 1, 2026)
- Works immediately with Claude Code, OpenCode, Codex

## Quick Start Path
1. Install Ollama (ollama.com)
2. Run `ollama launch claude` or `ollama launch opencode`
3. Pick a model (qwen3-coder recommended for coding)
4. Start coding with local AI - no API keys, no cloud
