# Article Outline: 12 Open Source SaaS Alternatives

## 1. Your SaaS Bill Is Eating Your Runway (Intro/Hook)
- Average SaaS spend per employee: $1K-$3.5K/year
- Equity-backed startups spend 107% of ARR on operations
- Real savings example: $70K-$330K/year for 50-person startup
- Frame: these are battle-tested, not toy projects

## 2. Plane (Project Management)
- Replaces Jira/Linear/Monday at $0 self-hosted
- 38K+ GitHub stars, sprints, issues, wiki, GitHub integration

## 3. PostHog (Product Analytics)
- Replaces Amplitude + LaunchDarkly + Hotjar
- Free tier: 1M events/month, 98% of users pay nothing
- Used by Airbus, ElevenLabs

## 4. Keycloak (Authentication)
- Replaces Auth0/Okta
- Red Hat backed, CNCF project
- SSO, social login, RBAC, MFA, passkeys
- Phase Two for managed hosting

## 5. Strapi (Headless CMS)
- Replaces Contentful/Sanity
- 71K GitHub stars, JS/TS, REST + GraphQL
- Used by Amazon, Airbus

## 6. SigNoz (Observability)
- Replaces Datadog/New Relic
- OpenTelemetry native, ClickHouse backed
- Up to 80% cost savings vs Datadog

## 7. Rocket.Chat (Team Communication)
- Replaces Slack/Teams
- 43K stars, federation, E2E encryption, GDPR
- Used by US Navy, Deutsche Bahn

## 8. Listmonk (Email/Newsletters)
- Replaces Mailchimp/ConvertKit
- Go binary, handles millions of subscribers
- ~$0.10/1000 emails with Amazon SES

## 9. Formbricks (Surveys/Feedback)
- Replaces Typeform/SurveyMonkey
- In-app, website, link, email surveys
- GitHub Accelerator cohort member

## 10. MinIO (Object Storage)
- Replaces Amazon S3 (self-hosted)
- 59K stars, S3 compatible, 2.6Tbps reads
- CAVEAT: Community edition in maintenance mode since late 2025

## 11. Gitea + Woodpecker CI (Code Hosting + CI/CD)
- Replaces GitHub/GitLab hosted + CircleCI
- Runs on Raspberry Pi, near GitHub Actions compatibility
- Codeberg runs on Gitea

## 12. NocoDB (Database/Spreadsheet)
- Replaces Airtable
- 60K stars, connects to existing databases
- Grid, kanban, calendar, gallery views

## 13. Upptime (Status Page/Monitoring)
- Replaces Pingdom/StatusPage
- Zero cost, runs on GitHub Actions/Issues/Pages
- 5-minute check intervals

## 14. Building a Stack That Grows With You (Wrap-up)
- Total savings: $15K-$50K (10 people), $70K-$330K (50 people)
- Start with 2-3 swaps, not all 12
- Self-host vs managed: be honest about DevOps capacity
- Owning tools = competitive advantage
