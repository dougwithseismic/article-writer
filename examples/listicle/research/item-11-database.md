# Research: NocoDB - Database/Spreadsheet

## Key Facts
- 60K GitHub stars (reached Jan 2026)
- Turns MySQL, PostgreSQL, SQL Server, SQLite into spreadsheet interface
- Views: grid, kanban, calendar, gallery
- API support (REST) for automation and integration
- No-code interface for non-technical users
- Handles millions of rows
- Team plan starts at $228
- Free self-hosted option

## Airtable Comparison
- Airtable: $20/user/month (Pro), limited records per base
- NocoDB: free (self-hosted) or much cheaper managed
- NocoDB connects to existing databases (Airtable does not)
- No vendor lock-in with NocoDB

## Trade-offs
- Fewer pre-built templates than Airtable
- Automation features less mature
- UI polish behind Airtable
