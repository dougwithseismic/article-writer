# Research: Rocket.Chat - Communication

## Key Facts
- 43K+ GitHub stars, biggest open-source comms project
- Used by Deutsche Bahn, US Navy, Credit Suisse, City of Cologne, eu-LISA
- Government adoption: NIPRNet, SIPRNet, JWICS, DoD ATO up to IL6
- GDPR compliant, E2E encryption
- Federation support for cross-organization collaboration
- Self-hosted or managed hosting
- Omnichannel: email, SMS, social media integration
- Video conferencing built-in

## Trade-offs
- UX still behind Slack's polish
- Self-hosting requires infrastructure management
- Some features locked to enterprise tier
