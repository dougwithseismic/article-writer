# Research: Strapi - Headless CMS

## Key Facts
- Leading open-source headless CMS, 100% JavaScript/TypeScript
- 71.2K GitHub stars, 20M+ downloads
- 3K+ companies in production (incl. PostHog, CodeRabbit, Amazon, Airbus)
- Strapi v5 released, up to v5.31.3 (Dec 2025)
- Features: REST + GraphQL APIs, i18n, RBAC, custom fields, TypeScript
- Supports PostgreSQL, MySQL, MariaDB, SQLite
- Self-hostable or Strapi Cloud (managed)
- Plugin ecosystem, SSO/Audit Logs/Review Workflows in Enterprise
- 10 years old (est. 2015)

## Trade-offs
- Enterprise features (SSO, audit logs) require paid plan
- Can be resource-heavy for large datasets
- v5 migration from v4 requires effort
