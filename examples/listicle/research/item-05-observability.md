# Research: SigNoz - Observability

## Key Facts
- Open-source Datadog/New Relic alternative
- Native OpenTelemetry support
- Unified logs, traces, metrics in one tool
- Built on ClickHouse (used by Uber, Cloudflare)
- Claims up to 80% cost savings vs Datadog
- All metrics: $0.1/million samples (no "custom metrics" premium)
- Unlimited hosts and team members
- Cloud, Enterprise (self-hosted), and Community (open source) editions
- LLM observability features
- Migration tool available for Datadog switch

## Trade-offs
- Smaller ecosystem of integrations vs Datadog
- Self-hosted requires ClickHouse expertise
- Newer product, community still growing
