# Research: Formbricks - Surveys/Feedback

## Key Facts
- Open-source experience management / survey platform
- Privacy-first, AGPLv3 license
- Survey types: in-app, website, link, email
- NPS, CSAT measurement at any customer journey stage
- Event-based triggers, fine-grained user segmentation
- Integrations: Slack, Zapier, HubSpot
- Self-hostable via Docker, also offers GDPR-compliant cloud
- Part of GitHub Accelerator inaugural cohort
- Free for personal and commercial use
- Positions as Qualtrics/Typeform alternative

## Trade-offs
- Smaller template library than Typeform
- Feature set still growing
- Smaller community than established competitors
