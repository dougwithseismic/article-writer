# Research: Plane - Project Management

## Key Facts
- 38.6K+ GitHub Stars, #1 open-source project management on GitHub
- 500K+ Docker pulls, 100+ contributors
- Replaces Jira, Linear, Monday, ClickUp
- Features: issues, cycles/sprints, modules/epics, docs/wikis, GitHub integration
- Launched in 2023, available cloud and self-hosted
- Streamlined UX, customizable statuses/labels/roles
- Commercial Edition expanding to DigitalOcean, Heroku, AWS AMI, Railway
- AI-powered assistance features

## Trade-offs
- Newer project, smaller plugin ecosystem than Jira
- Enterprise features still maturing vs Linear
